## Run Locally

Clone the project

```bash
  git clone https://github.com/deepaksah1304/NostalgicISM.git
```

Go to the project directory

```bash
  cd NostalgicISM_Backend/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon
```

Again go to the project directory

```bash
  cd NostalgicISM_frontend/
```

Install dependencies

```bash
  npm install
```

Start the client

```bash
  npm start
```
